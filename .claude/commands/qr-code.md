Génère un QR code pour Blanc Cocon.

## Consignes
1. Demande l'URL cible si non précisée : $ARGUMENTS
2. Génère le QR code en PNG haute résolution (box_size=40) pour impression avec la librairie Python qrcode dans /tmp/qr-venv/
3. Génère aussi une version web (box_size=20)
4. Sauvegarde les fichiers dans le dossier racine du projet
5. Affiche le QR code pour vérification
6. Propose de l'ajouter dans qr-avis-google.html ou qr-print.html si pertinent

Déploie les modifications du site Blanc Cocon sur GitHub Pages.

## Consignes
1. Fais un git status pour voir les fichiers modifiés
2. Montre un résumé des changements
3. Demande confirmation avant de commiter
4. Commit avec un message clair en français
5. Push sur main
6. Le site est sur : https://blanc-cocon.github.io/website/
